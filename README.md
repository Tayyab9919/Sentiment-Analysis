# Sentiment-Analysis
This dataset contains annotated customer reviews labeled for sentiment analysis.

### Labels:
0 = Negative  
1 = Positive  
2 = Mixed/Neutral  

### Annotation Strategy:
- Overall sentiment was considered
- Reviews with both positive and negative opinions were marked as Mixed

### Example:
"I like the product but it's overpriced" → 2 (Mixed)

### Total Samples:
15 reviews
